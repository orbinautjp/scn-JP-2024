# scn-JP-2024

Orbiter Space Flight Simulator 2024のシナリオファイルを日本語化するプロジェクトです。

https://github.com/orbinautjp/scn-JP-2024

## 対応バージョン

Orbiter2024で利用できます。

## インストール

ScenariosフォルダとHtmlフォルダの2つをコピーする。  
Orbiterのインストールフォルダを開いて、コピーしたフォルダを貼り付ける。  
既存のScenariosフォルダとHtmlフォルダを上書き。

## 利用規約

Orbiter本体の利用規約に準じているため、営利利用はできません。  
複製・改変は自由ですが、配布する場合は元ファイルへのリンクを添付してください。  
改変した場合は変更点を明記してください。

## 注意点

### Windows以外の環境

MacおよびLinuxでWine等を使って動作させている場合、日本語ファイルが正常に表示されない可能性があります。  
作者の環境では検証できないので、自力での対応をお願いします。

### ゲームのUI

MFDなどのUIは日本語に対応していません。

### ゲーム内字幕

リプレイ等で表示される字幕は日本語に対応していません。

### チェックリスト

Checklistsフォルダのゲーム内ヘルプ（Alt+F1で開きます）は翻訳していません。

「Orbiterの付属マニュアルやチェックリストを読んでも肝心なことが書いてない」という不満が、この活動を始めた原点なので、今後も触れることはないと思います。  
打ち上げやランデブーについては以下のページで解説しているので、参考にしてください。

https://orbinautjp.github.io/tutorial/
